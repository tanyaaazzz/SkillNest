# 🚀 SkillNest

**SkillNest** is a modern, intuitive learning platform designed to help users discover, explore, and master new skills through a seamless and engaging experience.

---

## ✨ What you can do

* 🔍 Discover and search courses effortlessly
* 📚 Explore detailed course content and structure
* 🎥 Preview lectures before enrolling
* 📊 Track your learning progress visually
* 🎓 Manage all your enrolled courses in one place

---
## ⚡ Tech Stack

**Frontend**
React.js • Tailwind CSS • React Router DOM

**State Management**
Context API

**Integrations & Libraries**
React YouTube (react-youtube) • Clerk Authentication


---

## 💡 Vision

SkillNest aims to simplify learning by combining **clean design, structured content, and smart progress tracking** —making skill development more accessible and engaging.
